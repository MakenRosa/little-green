# little-green

ESG-focused web application created as an academic project and maintained as a front-end architecture case.

## PT-BR resumo
Aplicacao web com tema ESG desenvolvida no contexto academico para praticar componentizacao em React, organizacao de layout e navegacao em single-page app.

## Highlights
- Component-based React structure
- Responsive layout and themed sections
- Educational project evolved as portfolio artifact

## Stack
- React 18
- React Router
- React-Bootstrap
- Sass modules

## Run locally
```bash
npm install
npm start
```
